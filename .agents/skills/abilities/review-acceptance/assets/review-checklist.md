# 受入レビュー チェックリスト

## Critical

| ID | 確認項目 |
|----|----------|
| C-01 | 必須 AC のいずれかが未検証または「いいえ」で未達理由がない |
| C-02 | 必須 FR に断絶があり、フォローアップがない |
| C-03 | `reviews/code.md` に Critical が残っているのに受入判定が合格 |
| C-04 | 権限マトリクスの検証が空 |

## Suggestion

| ID | 確認項目 |
|----|----------|
| S-01 | トレーサビリティ表にテスト列が空の必須 FR がある |
| S-02 | 将来 FR の移管記録がない（該当 FR がある場合） |

## 判定ルール

- Critical が1件でもあれば **不合格**
- Suggestion のみなら **条件付き合格**
- 指摘なしなら **合格**
